사용자의 편의성과 최신 모바일 UX 트렌드를 고려했을 때, 알림 시간 설정은 **페이지(Page)보다는 바텀 시트(Bottom Sheet) 형태의 모달**로 구현하는 것을 강력하게 추천합니다.

**추천 이유:**
1.  **컨텍스트 유지:** 설정 페이지를 벗어나지 않고 값을 변경한다는 느낌을 주어 사용자가 길을 잃지 않습니다.
2.  **가벼운 조작:** 시간 변경은 단일 항목 수정입니다. 이를 위해 새 페이지로 이동하고 다시 뒤로 돌아오는 과정은 불필요한 뎁스(Depth)를 추가합니다.
3.  **손가락 동선:** 바텀 시트는 화면 하단에 위치하여 엄지손가락으로 쉽게 시간을 조절하고 완료할 수 있습니다.

### 🎨 알림 시간 설정 UI/UX 상세 계획

**"간단하고 명확하게"**라는 요구사항에 맞춰, 설정 페이지의 리스트 항목 디자인부터 바텀 시트까지 연결되는 흐름을 설계했습니다.

#### 1. 진입점: 설정 페이지 내 리스트 (Entry Point)

기존 설정 페이지의 '알림' 섹션 UI를 조금 더 정보성 있게 개선합니다.

* **현재 UI:** `(아이콘) 복용 알림 [토글]`
* **변경 UI:** `(아이콘) 복용 알림` + `오후 10:00 >` (우측 텍스트 버튼)
    * **토글 스위치 제거 (선택 사항):** 알림을 끄고 싶은 경우, 바텀 시트 안에서 '알림 끄기'를 제공하거나, 시간 옆에 스위치를 유지하되 **시간 텍스트를 터치하면** 설정 모달을 띄웁니다.
    * **상태 표시:** 현재 설정된 시간이 우측에 주황색(포인트 컬러) 텍스트로 표시되어, 굳이 눌러보지 않아도 내 설정을 알 수 있게 합니다.

#### 2. 설정 모달: 바텀 시트 (Bottom Sheet)

사용자가 시간을 눌렀을 때 화면 하단에서 올라오는 반(Half) 높이의 시트입니다.

* **구성 요소 (위에서 아래로):**
    1.  **헤더 (Header):**
        * 중앙 타이틀: "알림 시간 설정" (Bold)
        * 우측 상단: '닫기' 또는 'X' 아이콘 (회색)
    2.  **타임 피커 (Time Picker):**
        * **iOS/Android 네이티브 피커**를 사용합니다. (가장 익숙하고 오류가 적음)
        * 큼직한 휠(Wheel) 방식이나 시계 다이얼 방식으로, 현재 설정된 시간이 기본으로 선택되어 있습니다.
    3.  **안내 문구 (Helper Text):**
        * 피커 바로 아래에 작은 글씨로 안내를 줍니다.
        * ℹ️ *"알림은 복용 주기에 맞춰 **복용일에만** 울립니다."* (사용자가 매일 울리는지 헷갈리지 않게 명시)
    4.  **저장 버튼 (CTA Button):**
        * `[ 설정 완료 ]`
        * **스타일:** 앱의 메인 컬러인 **주황색 배경** + 흰색 텍스트.
        * **인터랙션:** 누르면 시트가 내려가고, 변경된 시간이 즉시 반영되며 로컬 알림이 재예약됩니다.

#### 3. 사용자 시나리오 (UX Flow)

**Case A: 무료 유저 (잠금 상태)**
1.  설정 페이지에서 '복용 알림' 행을 터치합니다.
2.  시간 설정 모달 대신 **"프리미엄 결제 유도 페이지(Paywall)"**가 뜹니다.

**Case B: 프리미엄 유저 (활성 상태)**
1.  설정 페이지에서 '오후 10:00' 텍스트를 터치합니다.
2.  **바텀 시트**가 부드럽게 올라옵니다.
3.  시간을 '오전 8:00'로 변경하고 `[설정 완료]` 버튼을 누릅니다.
4.  시트가 닫히고 토스트 메시지(Toast)가 뜹니다. *"알림 시간이 오전 8시로 변경되었어요."*
5.  설정 페이지 리스트의 시간이 '오전 8:00'로 업데이트되어 보입니다.

### 📱 디자인 디테일 (White & Orange Theme)

* **진입점 텍스트:**
    * 기본: 검정색 (`#222222`)
    * 시간 표시(Value): **주황색** (`#FF6B00`) - 터치 가능하다는 힌트(Affordance) 제공.
* **바텀 시트:**
    * 배경: **깨끗한 흰색** (`#FFFFFF`)
    * 모서리: 상단 좌우 둥글게 (Radius 20px)
* **안내 문구:**
    * 중간 회색 (`#888888`)으로 시각적 소음을 줄임.

이 계획대로라면 사용자는 "몇 시에 알림이 오는지" 바로 알 수 있고, "복용일에만 온다"는 규칙도 자연스럽게 인지하게 됩니다. 구현하시겠습니까?
