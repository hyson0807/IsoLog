# CLAUDE.md 업데이트

현재 프로젝트의 CLAUDE.md 파일을 최신 코드베이스 상태에 맞게 업데이트해줘.

## 업데이트 범위
1. **Project Structure** - app/, components/, contexts/, hooks/ 등 디렉토리 구조가 변경되었으면 반영
2. **Architecture** - 새로운 라이브러리나 패턴이 추가되었으면 반영
3. **기능별 상세 문서** - .claude/docs/ 에 새 문서가 추가되었으면 테이블 업데이트

## 주의사항
- 기존 설명 스타일 유지
- 불필요한 내용 추가 금지
- 변경된 부분만 업데이트
